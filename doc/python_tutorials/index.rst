FlowCal's Python API Tutorials
==============================

The ``FlowCal``'s Python API tutorials are listed below in order of increasing complexity. We recommend you to go through them in order.

.. note:: The ``FlowCal`` Python API tutorials assume that the reader is familiar with Python, ``numpy`` and ``matplotlib``. It also assumes that the reader has Python 2.7 installed, as well as ``FlowCal`` and all its dependencies. For more information on installation, refer to the :doc:`Installation</installation>` section.
