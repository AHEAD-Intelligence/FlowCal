FlowCal's Python API Tutorial
=============================

The different sections of the ``FlowCal``'s Python API tutorial are listed below in order of increasing complexity. We recommend the reader to go through them in order.

.. note:: The ``FlowCal`` Python API tutorial assumes that the reader is familiar with Python, ``numpy`` and ``matplotlib``. It also assumes that the reader has Python 2.7 installed, as well as ``FlowCal`` and all its dependencies. For more information on installation, refer to the :doc:`Getting started</getting_started/index>` section.
